시멘트 마크업(semantic_markup)

시멘틱(semantic)의 사전적 뜻은 '의미론적인' 정도로 해석할 수 있는데, 이것은 마크업을 할 때 의미에 부여하는 태그를 사용하라는 뜻입니다.


HTML 문서는 보통 시각적인 방법으로 사용자들에게 정보를 전달하지만, 모든 사람들이 정보를 동일하게 받아들일 수 있는 환경이 아닐 수도 있어서, 이런 경우에 전달하려는 태그의 의미가 적절해야 합니다.

[예시1]

<!DOCTYPE html>
<html lang="ko">
<head>
　　<meta charset="UTF-8">
　　<title>시멘틱 마크업</title>
</head>
<body>
　　반응형웹
</body>
</html>

"반응형웹"

이때 반응형 웹 부분을 마크업하면 제목 부분인 것을 알 수 있게 됩니다.

[예시2]

<!DOCTYPE html>
<html lang="ko">
<head>
　　<meta charset="UTF-8">
　　<title>시멘틱 마크업</title>
</head>
<body>
　　<h1><반응형웹></h1>
</body>
</html>

"반응형웹"
