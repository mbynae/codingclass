비구조화 할당

비구조화할당이란 배열이나 객체의 속성 혹은 값을 해체하여 그 값을 변수에 각각 담아 사용하는 자바스크립트 표현식입니다.

▶ 문법

const {키1, 키2, 키3} = {값1, 값2, 값3}; = 변수 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const {a, b, c} = obj;

    document.write(a);
    document.write(b);
    document.write(c);
}

객체구조분해할당

구조 분해 할당이란 배열이나 객체의 속성을 해체하여 그 값을 개별 변수에 담을 수 있게 하는 자바스크립트 표현식입니다.

▶ 문법

const {키1: 변수1, 키2: 변수2, 키3: 변수3} = {값1, 값2, 값3}; = 변수 

{
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    const {a: name1, b: name2, c: name3} = obj;

    document.write(name1);
    document.write(name2);
    document.write(name3);
}
