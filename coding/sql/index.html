MYSQL

MySQL은 데이터베이스 소프트웨어입니다. 일반적으로 데이터를 추가하거나 검색, 추출하는 기능을 포함해서 데이터베이스라고 부릅니다.

MySQL은 세계에서 가장 많이 쓰이는 오픈 소스 관계형 데이터베이스 관리시스템(RDBMS)입니다. MySQL은 PHP 스크립트 언어와 상호 연동이 잘 되면서 오픈소스를 개발된 무료 프로그램입니다. 그래서 홈페이지나 쇼핑몰(워드프레스, cafe24, 제로보드, 그누보드) 등 일반적으로 웹 개발에 널리 사용하고 있습니다.

MYSQL 설치

MAMP란 웹사이트를 개발할 때 쓰이는 기술 스택인 maxOS, Apache, MySQL, PHP 의 약어이자 솔루션 스택이다.
https://www.mamp.info/en/downloads/

테이블

테이블 만들기

create table 테이블 이름; 
create table member (
    myMemberID int(10) unsigned auto_increment,
    youEmail varchar(40) NOT NULL,
    youName varchar(20) NOT NULL,
    youPass varchar(20) NOT NULL,
    youBirth int(20) NOT NULL,
    youAge int(20) NOT NULL,
    regTime int(20) NOT NULL,
    PRIMARY KEY (myMemberID)
) charset=utf8;

show tables;

이름 중에 '김'으로 시작하는 멤버 출력

SELECT * FROM member WHERE youName LIKE '김%';

이름 중에 '김'씨 이거나 '황'씨인 경우

SELECT * FROM member WHERE youName LIKE '김%' OR youName LIKE '황%';

이름 중간에 '상'이 있는 멤버 출력

SELECT * FROM member WHERE youName LIKE '_상_';

또는 연으로 끝나는 경우 
SELECT * FROM member WHERE youName LIKE '__연';

이름 중에 데이터가 없는 경우 출력

SELECT * FROM member WHERE youName IS NULL;

반대의 경우
SELECT * FROM member WHERE youName IS NOT NULL;

멤버 중에 나이가 18세 이상이면서 김씨인 멤버를 출력

SELECT * FROM member WHERE youAge >= 18 AND youName LIKE '김%';

멤버 중에 나이가 18세 이상이거나 김씨인 멤버를 출력

SELECT * FROM member WHERE youAge >= 18 OR youName LIKE '김%';

멤버 중에 나이가 20세 이상이면서 아이디가 2 또는 10인 멤버

SELECT * FROM member WHERE (myMemberID = 2 OR myMemberID = 10) AND youAge >= 20;

회원 데이터 5개만 불러오기(1~5)

SELECT * FROM member LIMIT 5;

회원 데이터 5개만 불러오기(6~10)

SELECT * FROM member LIMIT 5, 5;

회원 이메일을 기준으로 5개만 불러오기(1~5)

SELECT youMail FROM member LIMIT 5;

테이블 수정하기

UPDATE 테이블명(필드명) SET 컬럼명 조건

모든 회원의 핸드폰 번호 변경

UPDATE member SET youPhone = "010-0000-0000";

아이디가 3번인 회원의 비밀번호를 '0123'

UPDATE member SET youPass = "0123" WHERE myMemberID = 3;

회원 아이디 5~10번까지 비밀번호를 "1234567"

UPDATE member SET youPass = "1234567" WHERE myMemberID BETWEEN 5 AND 10;

비밀번호를 "1234567"로 변경하고 내림차순으로 10개만 변경하기

UPDATE member SET youPass = "1234567" ORDER BY youPass DESC LIMIT 10;

테이블 삭제하기

DELETE FROM 테이블명 조건

회원 아이디가 5번인 멤버 삭제

DELETE FROM member WHERE myMemberID = 5

회원 비밀번호가 '1234'인 멤버 삭제

DELETE FROM member WHERE youPass = '1234';

회원 이름을 기준으로 5번째까지 멤버 삭제

DELETE FROM member ORDER BY youName LIMIT 5;

지정한 멤버 삭제

DELETE FROM member WHERE myMemberID IN (7, 8, 9);

ALTER TABLE member ADD youGender enum('m', 'w', 'x') default 'x' comment "남자 m, 여자 w" AFTER youPhone;

SQL 및 PHP 속성 정리

SQL과 PHP에서 사용되는 속성입니다.
  • UNSIGNED : 칼럼내에서 음수를 포함하지 않거나 혹은 수의 range를 양수쪽으로 더 넓게 가지고싶을때 사용
  • - AUTO_INCREMENT : 테이블 자동 증가
    자동증가 속성이 부여된 열은 새 행이 추가될 때마다 값이 1씩 자동으로 증가합니다.
    자동 증가 값은 테이블 행의 삭제와 무관하게 새 행이 추가되면 값이 계속 증가합니다.
  • int( ) : 숫자값. 괄호안에 숫자를 넣으면 값의 자리 범위 설정
  • varchar( ) : 문자값. 괄호안에 숫자를 넣으면 값의 자리 범위 설정(가변 길이)
  • PRIMARY KEY : 기본값. 절대로 중독될 수 없는 값을 정함
  • enum : 정해진 특정한 값만 선택해 적을 수 있도록 하는 타입
  • DEFAULT : 데이터 입력시 값을 빠트릴 때 정해진 기본값을 사용함
  • ALTER TABLE : 테이블을 수정할 때 사용하는 공식
  • ORDER BY : 데이터를 정렬할 때 사용(ex. 오름차순, 내림차순)
  • SELECT LIMIT : 위에서부터 지정한 갯수만큼 선택 (LIMIT의 앞 숫자는 불러올 갯수, 뒷 숫자는 몇번부터 불러올지 선택 LIMIT 5, 5는 6~10번을 불러옴)
  • LIKE : 문자열의 패턴을 검색 (%와 _을 사용할 수 있음)
  • TINYTEXT : 최대 255 글자의 범위
  • num_rows : 데이터 열의 수를 반환합니다. (일치하는 수가 없으면 0, 있으면 갯수만큼 표시)
  • exit : 실행문을 즉시 종료
  • -filter_var($email, FILTER_VALIDATE_EMAIL) : 이메일 주소를 입력받았을 때 입력받은 이메일이 이메일 주소 형식에 맞게 입력했는지 확인하기 위해서 사용하는 함수
  • fetch_array( ) : 데이터의 결과를 배열로 반영
  • MYSQLI_ASSOC : row["password"] 와 같이 컬럼명으로 값을 불러옴
  • MYSQLI_NUM : row["0"] 와 같이 인덱스로 값을 불러옴
    ($row = $result->fetch_array(MYSQLI_BOTH); 이렇게 쓰면 둘 다 사용 가능
  • var_dump( ) : 변수의 정보를 출력하는 함수
  • Header(location: (주소) ) : 헤더 페이지 이동 함수. 조건 충족 시 location에 써있는 페이지로 이동함(예를들어 로그인 시 메인페이지로 이동)(리다이렉트)
  • location.href = “이동할 페이지 주소” : 작성된 페이지로 이동하도록 함(자바스크립트 리다이렉트)
    둘 중 아무거나 사용해도 됨
  • sesstion start( ) : 데이터를 세션에 저장하기 위해 사용하는 함수
  • $_SESSION[’세션 이름’] = “세션값” : 세션으로 세션값 데이터를 저장함
  • $VALUE = $_SESSION[’세션이름’] : 세션에 저장된 세션값 데이터를 가져옴
  • unset( ) : php의 변수 제거에 사용되는 함수
  • !$변수 : 변수 앞에 !가 붙을 시 !는 Not의 의미를 가짐
  • isset( ) : 함수 안에 적힌 변수가 설정되어 있는지 여부를 true와 false로 반환해줌
  • escape ( ) : 저장되어있는 정보를 사용자에게 노출할 때 발생할 수 있는 문제를 차단하는 행위
  • real_escape_string( ) : 입력 값에 “DROP TABLE 테이블명 ” 같은 데이터가 수정 될 수 있는 문장을 방지함. 인자값들을 필터링된 인자로 받아줌
  • ceil / floor / round : 각각 값을 올림 내림 반올림
  • date() 함수 : string date ( string $format, [int $timestamp] ) : 정수형으로 주어지는 timestamp나, timestamp가 주어지지 않았을 경우에는 현재 로컬 시간을 사 용하여, 주어진 포맷 문자열에 따라 형식화한 문자열을 반환합니다. 즉 timestamp는 선택적이고, 기 본값은 time()의 값입니다. 지원되지 않는 포맷 문자는 그대로 출력됩니다.
