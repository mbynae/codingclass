2. font-size 단위

문자의 크기를 지정하는 속성입니다.
글자 크기는 기본 16px이며 다음과 같은 속성이 있습니다.

속성 내용
px 해상도에 따라 상대적으로 달라지는 기본 단위입니다.
% 부모 요소의 글자 크기를 100% 기준으로 계산한 % 단위입니다.
em 부모 요소의 글자 크기를 100% 기준으로 계산한 100분의 1단위입니다.
rem em과 비슷하지만 부모 요소가 아닌 최상위 요소의 크기를 기준으로 하는 100분의 1단위입니다.
vw 뷰포트 너비값의 100분의 1단위입니다.
vh 뷰포트 높이값의 100분의 1단위입니다.
